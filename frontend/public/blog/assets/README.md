# Ảnh blog

Để file ảnh dùng trong bài viết ở đây. Tự phục vụ tại `/blog/assets/<tên-file>`.

- **Ảnh bìa bài** (hero + ảnh chia sẻ riêng): frontmatter `image: "/blog/assets/<slug>.jpg"` — nên ~1200×630, `.webp`/`.jpg`.
- **Ảnh trong thân bài**: Markdown `![mô tả ảnh](/blog/assets/<tên>.jpg)`.
- `alt` (mô tả) phải có nội dung → tốt cho SEO + người khiếm thị.

⚠️ Nguồn & kiểm duyệt ảnh: xem `frontend/content/README.md` mục 8.
